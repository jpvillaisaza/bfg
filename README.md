# The BFG

The BFG is a ~~big~~ friendly ~~giant~~ books API...

```
$ git clone git@github.com:stackbuilders/bfg.git
$ cd bfg/
```

```
$ stack setup
```

```
$ stack build --flag bfg:init --exec bfg-api
```

```
$ stack build --exec bfg-api
```
